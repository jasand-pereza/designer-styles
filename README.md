# Designer Styles

This experimental tool is intended to let a web designer style a basic page template. This page template has already been created with common pages elements. 

## Included DOM elements - not a complete list

* h1 - h6
* p
* strong
* b
* a
* blockquote
* sup 
* ul
* ol
* li
* img
* cite
